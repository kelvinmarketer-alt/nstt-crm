✈️ Telegram Bot 2 chiều — Shipper

Shipper thao tác đơn ngay trên Telegram, không cần app riêng

📡 Webhook URL (setup bot)

Để bot nhận tin nhắn của shipper, bạn cần đặt webhook trên Telegram:

  1. Deploy 1 endpoint nhận webhook (Cloudflare Workers / Vercel / Netlify Functions) — file mẫu: tg-webhook.example.js
  2. Gọi API set webhook:
    curl -F "url=https://your-domain.com/tg-webhook" \\
      https://api.telegram.org/bot<BOT_TOKEN>/setWebhook
  3. Bot sẽ nhận message → parse command → gọi API NSTT cập nhật STORE
💡 Bản demo này chỉ simulate command — không thực sự nhận webhook. Để chạy thật cần deploy backend.

⚡ Commands hỗ trợ

/giao <mã đơn>

Đánh dấu đơn đã giao thành công cho KH
/giao NSTT-000142
→ Status: transit → delivered. Tự update KPI shipper + cộng doanh thu KH.

/lay <mã đơn>

Đã lấy hàng ở kho, bắt đầu giao
/lay NSTT-000142
→ confirmed → pickup → transit

/hoan <mã> <lý do>

KH từ chối/không có mặt — đánh dấu hoãn
/hoan NSTT-000142 KH bận, hẹn 14h chiều

/pod <mã đơn>

Gửi kèm ảnh POD (Proof of Delivery)
/pod NSTT-000142 + ảnh đính kèm
→ Ảnh lưu trong STORE.pod_photos, có thể xem ở trang Returns & POD.

/donhom — Xem đơn hôm nay

Bot trả về list đơn chưa giao của shipper
/donhom
